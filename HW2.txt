=========
Debugging
=========
1. Which of the following does an issue-tracking system not do?
    A. Provides visibility to the debugging effort
    B. Enables the tracking and planning of releases
    C. Automatically fix reported issues.
    D. Facilitates the prioritization of work items
    Correct Answer: C
2. Which of the following is not a basic step in debugging?
    A. Recognize that a bug exists
    B. Determine a fix for the bug
    C. Isolate the source of the bug
    D. Turn off PC.
    Correct Answer: D
3. Which of the following are not a step in the Feynman Algorithm?
    A. Think real hard
    B. Fix the problem
    C. Write down the solution
    D. Write down the problem
    Correct Answer: B
